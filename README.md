# homebrew-fastcut

Homebrew tap for [FastCut](https://github.com/7757/FastCut) — a lightweight macOS clipboard-history manager.

```sh
brew install --cask 7757/fastcut/fastcut
```

FastCut is self-signed (not notarized); if the first launch is blocked, right-click → Open,
or add `--no-quarantine` to the command above.
